## Estructura de Directorios

- **_dog-images_**: Contiene las imágenes de las 70 razas de perros, organizadas en carpetas para _train_, _test_ y _validation_.
- **_gradio-testimages_**: Incluye imágenes etiquetadas, utilizadas para probar manualmente los modelos entrenados mediante la interfaz de _Gradio_.
- **_development_**: Carpeta donde se encuentran los _Jupyter Notebooks_ correspondientes a las distintas etapas del trabajo.
- **_gradio-interface_**: Aquí se encuentra la interfaz de Gradio desarrollada para la etapa 2, que permite probar los modelos _ResNet50_, _ResNet18_ y la _CNN custom_.
